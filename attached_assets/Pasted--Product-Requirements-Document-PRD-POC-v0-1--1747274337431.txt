
# 📄 Product Requirements Document (PRD)

**프로젝트명**: 시놉시스 기반 영상 제작 자동화 POC  
**버전**: v0.1  
**작성일**: 2025-05-15  
**작성자**: (당신)

---

## ✅ 1. 개요

### ● 목적
- 사용자가 입력한 시놉시스를 자동으로 씬(scene) 단위로 분해하고,
- 각 씬에 대해 이미지 프롬프트 및 시각 자료를 생성하며,
- 생성된 이미지를 바탕으로 영상 프롬프트와 negative prompt를 자동 생성.
- 최종적으로 영상 제작 자동화 파이프라인의 기반을 마련하는 것이 목적입니다.

> 본 문서는 **로그인/저장 기능 없이 동작하는 POC** 화면 구현을 목적으로 합니다.

---

## ✅ 2. 핵심 기능 요약

| 단계 | 기능 설명 |
|------|-----------|
| 1 | 시놉시스 입력 |
| 2 | 시놉시스를 씬으로 분해 (5~20개) |
| 3 | 씬 편집 및 순서 재배치 |
| 4 | 이미지 프롬프트 생성 |
| 5 | 이미지 생성 (썸네일) |
| 6 | 이미지 선택 |
| 7 | 이미지 디스크립션 (영상 프롬프트 & Negative Prompt 생성) |
| 8 | 프롬프트 다운로드(JSON) |

---

## ✅ 3. 사용자 흐름

[시놉시스 입력]  
↓  
[씬 분해 API 호출]  
↓  
[씬 리스트 확인 및 편집]  
↓  
[이미지 프롬프트 생성 → 이미지 생성 → 이미지 선택]  
↓  
[이미지 디스크립션 → 영상 프롬프트 생성]  
↓  
[프롬프트 확인 및 다운로드]

---

## ✅ 4. 화면 구성

### 1. 시놉시스 입력 화면

- 시놉시스 입력 텍스트박스
- 씬 개수 슬라이더 (5 ~ 20)
- "씬 분해" 버튼 → `/parse-scenes` API 호출

### 2. 씬 리스트 및 카드 뷰

- 가로 슬라이더 형식으로 카드 탐색
- 각 씬은 개별 카드로 구성됨
- 씬 내용 편집 가능
- 이미지 생성 및 디스크립션 결과 상태 표시

### 3. 씬 카드 구성

- 씬 텍스트 수정 영역
- "이미지 프롬프트 생성" 버튼 (`/generate-image-prompt`)
- 이미지 프롬프트 출력
- "이미지 생성" 버튼 (`/generate-image`)
- 이미지 썸네일 표시 (2~3장)
- 선택 가능 (✔️)
- "영상 프롬프트 생성" 버튼 (`/describe-image`)
- 영상 프롬프트 및 Negative Prompt 표시

### 4. 최종 출력

- 전체 씬 완료 상태 확인 (프로그레스 바)
- 프롬프트 리스트(JSON) 다운로드

## ✅ 5. 기술 스택

| 항목 | 선택 |
|------|------|
| 프론트엔드 프레임워크 | Vue 3 + Vite |
| 전역 상태 관리 | Pinia |
| UI 프레임워크 | Tailwind CSS |
| API 응답 시뮬레이션 | MSW (Mock Service Worker) |
| 배포 대상 | 로컬 or Vercel, Netlify 정적 사이트 |

---

## ✅ 6. API 명세 (Mock 버전 기준)

| API Endpoint | Method | Input | Output |
|--------------|--------|-------|--------|
| `/parse-scenes` | POST | `{ synopsis, scene_count }` | `[{ id, text }]` |
| `/generate-image-prompt` | POST | `{ scene_text }` | `{ prompt }` |
| `/generate-image` | POST | `{ prompt }` | `{ images: [url1, url2, ...] }` |
| `/describe-image` | POST | `{ image_url }` | `{ video_prompt, negative_prompt }` |

---

## ✅ 7. 비기능 요구사항

- 로그인: ❌ 없음  
- DB 연동: ❌ 없음  
- 로컬 상태 유지: ✔️ Pinia 활용  
- 오류 처리: 간단한 fallback 메시지  
- 반응형: 데스크탑 기준 (모바일 대응 선택 사항)  
- 렌더링 기능: ❌ POC에는 포함하지 않음  

---

## ✅ 8. 결과물 (POC 기준)

- 시놉시스 입력 → 씬 분해 → 이미지 생성 → 영상 프롬프트까지 전체 흐름 UI  
- MSW 기반의 Mock API를 통한 인터랙션 시뮬레이션  
- 사용자 정의한 JSON 파일 다운로드  
- 실제 이미지/영상 생성은 미포함
